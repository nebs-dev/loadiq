## ai-fitness-coach
AI-assisted fitness coaching platform that analyzes wearable data  (steps, heart rate, sleep and activity load) to provide personalized  training, recovery and body composition insights.

## Garmin Integration
The application integrates with Garmin Connect using the official Garmin Health API.

Users explicitly authorize access to their Garmin data through the Garmin consent flow.

The platform retrieves selected health metrics including:

• steps
• heart rate
• sleep metrics
• activity summaries

Data is used exclusively to generate personalized fitness insights for the user.

Users may revoke access at any time by disconnecting their Garmin account.

This application does not resell or redistribute Garmin user data.
