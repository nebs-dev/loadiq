# Privacy Policy

Last updated: March 2026

AI Fitness Coach is developed by Bimotech.

This application analyzes wearable fitness data to provide personalized coaching insights.

## Data Collection

With explicit user consent, the application may access selected health data from Garmin Connect, including:

- steps
- heart rate
- sleep metrics
- activity summaries
- body metrics if available

Users may also optionally provide:

- body weight entries
- progress photos
- nutrition information

## Purpose of Data Processing

The data is used exclusively to generate personalized insights about:

- training load
- recovery
- body composition trends
- fitness progress

## User Consent

Users explicitly authorize access to their Garmin data via the Garmin Connect authorization flow.

Users can revoke access at any time.

## Data Sharing

User data is **never sold or shared with third parties**.

Data is used solely to provide the service requested by the user.

## Data Deletion

Users may request deletion of their data at any time.
