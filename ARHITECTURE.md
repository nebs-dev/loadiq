# Architecture Overview

AI Fitness Coach analyzes wearable data and user inputs to generate personalized insights.

Data sources:
• Garmin wearable data
• User weight entries
• Optional nutrition data
• Optional progress photos

Processing pipeline:

1. Data ingestion from wearable APIs
2. Metric normalization
3. Trend detection (weight, recovery)
4. AI insight generation

The system provides daily and weekly coaching insights.
